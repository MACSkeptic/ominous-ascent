# ominous ascent

## before submitting a pull request

* make sure your code passes lint (grunt will tell you)
* make sure the code has tests
* make sure the build for the last commit in your pull request is green on travis
* squash all non-meaningful commits
* rebase your tree with master

## contributing

* create a [fork](https://github.com/MACSkeptic/ominous-ascent/fork_select)
* submit a [pull request](https://github.com/MACSkeptic/ominous-ascent/pulls)
